## Features ‚ú®

- Next.js 16 + React 19
~ Built on the newest app architecture with server components, streaming, and optimized performance.

- Day/Night mode
~ Powered by next-themes with smooth transitions.

- Dynamic Color Palettes
~ Fully customizable color palettes.

- Tailwind CSS 4
~ Uses the latest utility engine and tailwindcss-animate for polished motion.

- Radix UI Components
~ Accessible, unstyled primitives enhanced with custom design tokens.

- Framer Motion Integration
~ Fluid animations, interaction feedback, and scroll-based motion.

- Command Menu (cmdk)
~ Spotlight-like search and navigation for quick access across the site.

- 3D & Visual Sections Ready
~ Designed to support parallax, GSAP-style motion, and animated hero areas.

- Zod + React Hook Form
~ Fully type-safe forms with schema validation and elegant UI.

- Mobile-First + Responsive

## Install Prerequisites üì¶

- For Arch based distros:
```bash
sudo pacman -Syu git nodejs npm --noconfirm
```
- For Debian based distros:
```bash
sudo apt update
sudo apt install git nodejs npm -y
```
- For Fedora based distros:
```bash
sudo dnf install git nodejs npm -y
```
- For MacOS:
  - Install Homebrew (if not already installed) 
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
```bash
# If you are already have Homebrew
brew install git node
```
- For Windows:
  - Install Git and Node.js/NPM if not already installed
  - Git: https://git-scm.com/downloads
  - Node.js (includes npm): https://nodejs.org/en/download/
